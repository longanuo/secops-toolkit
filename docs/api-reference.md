# API Reference

This document will contain the API reference for the `secops-core`, `secops-offense`, and `secops-defense` modules.

## Modules

### secops-core
- `secops_core.config`: Configuration management.
- `secops_core.logger`: Logging utilities.
- `secops_core.mcp_server`: MCP Server functionality.

### secops-offense
- `secops_offense.attack_engine.AttackEngine`: Main engine for vulnerability scanning.

### secops-defense
- `secops_defense.evaluator`: Security evaluation components.
- `secops_defense.firewall`: Firewall rules management.
